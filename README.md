**Guess The Word** is a fun and interactive word-guessing game built with HTML, CSS, and JavaScript.
Try to guess the hidden 6-letter word within limited attempts. Use hints if needed, 
navigate using your keyboard, and restart the game anytime.

## Features

- 6-letter word guessing
- Limited tries with feedback:
  - Green: correct letter & position
  - Yellow: correct letter, wrong position
  - Gray: incorrect letter
- Hints system (2 hints per game)
- Keyboard navigation (Arrow keys + Backspace)
- Restart button to start a new game
- Responsive layout for desktop and mobile

## How to Play

1. Type letters in the inputs to guess the word.
2. Press **Check** to submit your guess.
3. Use the **Hint** button if stuck.
4. See the feedback for each letter.
5. Restart anytime using the **Restart** button.

## Tech Stack

- HTML
- CSS (Responsive)
- JavaScript (DOM manipulation)
